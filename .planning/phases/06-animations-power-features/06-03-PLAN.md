**IMPORTANT NOTE on breadcrumb positioning:** The breadcrumbs need to appear visually below the fixed header. Since `main > *` already has `padding-top: 4.5rem` in tailwind.css for header clearance, and the breadcrumbs sit outside `<main>`, add `pt-[4.5rem]` to the breadcrumbs nav to push it below the fixed header. However, this means the main content's padding-top will stack with the breadcrumbs' height. To handle this cleanly: place Breadcrumbs INSIDE `<main>`, just before the Suspense/AnimatedOutlet. This way, the existing `main > *` padding-top applies to the breadcrumbs as the first child, and the page content flows naturally after.
