3. Add FOUC prevention script to index.html -- insert BEFORE the Matomo script in <head>, as the FIRST script element:
   ```html
   <script>
     (function() {
       var theme = localStorage.getItem('darkMode');
       if (theme === 'true') {
         document.documentElement.classList.add('dark');
       } else if (theme === null) {
         if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
           document.documentElement.classList.add('dark');
         }
       }
     })();
   </script>
   ```
