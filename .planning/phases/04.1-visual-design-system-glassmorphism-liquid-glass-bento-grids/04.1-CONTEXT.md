# Phase 04.1: Visual Design System — Glassmorphism, Liquid Glass & Bento Grids - Context

**Gathered:** 2026-03-13
**Status:** Ready for planning

<domain>
## Phase Boundary

Apply a cohesive glassmorphism + liquid glass fusion aesthetic with claymorphism accents across all 9 pages. Replace tab-based tool navigation with editorial bento grid layouts that expand in-place. Add animated molecule cards with hover glow and expand/collapse. Implement resizable input/output split panels in expanded tool views. Add a 2-molecule comparison mode with a bottom tray and side-by-side view. All effects must work in both light and dark modes with mobile-optimized fallbacks.

</domain>

<decisions>
## Implementation Decisions

### Glass Aesthetic
- Bold glassmorphism: backdrop-blur 24px, bg white/10 (light) / slate-800/20 (dark), border 1px white/30, inner-shadow white glow
- Background gradient mesh visible through glass panels on ALL 9 pages
- Per-page color palettes: HomePage blue→purple, ChemPage sky→teal, ConvertPage indigo→blue, DepictPage violet→pink, OCSRPage emerald→cyan, ToolsPage amber→orange, AboutPage slate→blue, Privacy/Terms neutral gray
- Same glass intensity in both light and dark modes — shift color palette only (light: soft pastels, dark: deep jewel tones)
- Glass header: frosted glass bar, content scrolls behind it, backdrop-blur
- Glass footer: matches header aesthetic, frosted glass bar

### Liquid Glass Refraction
- Hover-activated refraction distortion on glass panels (SVG feTurbulence filter or CSS approach)
- At rest: standard frosted glass (clean, readable)
- On hover: refraction distortion appears, subtle warp/shift, slight scale (1.005), transition 300ms ease
- On leave: smooth transition back to standard glass
- No refraction on mobile — desktop only for performance

### Claymorphism Accents
- Raised interactive elements: buttons, toggles, interactive cards get soft puffy shadows (inner + outer)
- Tactile, pressable feel on all interactive controls
- Glass = background containers/panels, Clay = interactive elements
- Clear visual separation between glass surfaces and clay controls

### Bento Grid Layouts
- Editorial varied style: hero-sized cards (2x2 or full-width) for primary tools, medium cards for secondary, compact cards for minor tools
- Applied to tab-based tool pages: ChemPage (10 tools), ToolsPage (4 tools), ConvertPage, DepictPage
- Replaces current tab-switching navigation with visual grid overview showing all tools
- Also applied to HomePage feature cards
- Card content: label + description + lucide icon (no live previews, keep it simple)
- Click to expand in-place: card animates to fill grid area, other cards shrink/fade, tool input/output UI appears inside expanded card
- Framer Motion layoutId for expand/collapse with spring physics (stiffness:300, damping:25)
- Other cards animate to opacity 0.3, scale 0.95 during expansion
- Input/output UI fades in after expand completes

### Molecule Card Effects
- Hover: glow + lift effect — translateY -4px, soft colored glow border matching page gradient palette
- Expand to show details: click expands card to reveal additional molecule properties (MW, formula, descriptors, SMILES)
- Expanded card stays in same layout position, other cards adjust around it

### Resizable Panels
- All expanded tool views get resizable input/output split panels on wide screens (>1024px)
- Left panel: input (SMILES input, options, submit button)
- Right panel: output (structure visualization, properties, data tables)
- Draggable divider between panels
- Mobile (<1024px): stacked vertically (input above, output below)
- Panel positions persisted to localStorage per tool — user's preferred layout remembered

### Comparison Mode
- "Add to Compare" button on tool results
- Bottom comparison tray slides up from viewport bottom when first molecule is added (glass-styled)
- Tray shows added molecules with remove (✕) buttons and "Compare Side-by-Side" action
- Maximum 2 molecules for comparison
- Side-by-side view shows both molecules with their structures, properties, and descriptors in parallel columns
- Slide-up animation for tray appearance

### Mobile Behavior
- Reduced blur: backdrop-blur 12px on mobile (<768px) instead of 24px for GPU performance
- No SVG refraction effect on mobile
- Simpler gradients: 2-color gradient instead of full mesh on mobile
- Static glass (no hover animations) on touch devices
- Bento grid: single-column stack on mobile, hero cards go full-width
- Resizable panels: disabled on mobile, stacked vertically instead

### HomePage
- Glass hero card: large frosted glass panel over page gradient containing title, tagline, and CTA button
- Clay-styled CTA button inside hero panel
- Feature cards in bento grid layout below hero (editorial varied style)

### Transition Animations
- Bento expand/collapse: Framer Motion layoutId, spring physics (stiffness:300, damping:25)
- Comparison tray: slide up from bottom, glass-styled persistent bar
- Keep animations simple and consistent with existing motion/react patterns in the app
- Do not over-complicate — elegance over complexity

### Claude's Discretion
- Exact SVG filter parameters for liquid glass refraction effect
- Gradient mesh implementation technique (CSS radial-gradient composition vs canvas vs SVG)
- Exact clay shadow values (inner/outer shadow specifics)
- Bento grid tile size ratios per page (which specific tools get hero/medium/compact sizing)
- Resizable panel library choice (or custom implementation)
- Comparison tray state management approach
- Performance optimization details (will-change, GPU layers, etc.)
- About/Privacy/Terms page glass adaptation specifics

</decisions>

<specifics>
## Specific Ideas

- "Do not make anything too over complicated" — elegance and simplicity over flashy complexity
- "Avoid band-aids, shortcuts — this is a production system, accuracy first" (carried from Phase 1)
- Bold glass is the centerpiece visual — Apple Vision Pro / macOS Sequoia vibrancy as inspiration
- Liquid glass refraction (hover-only) differentiates from standard glassmorphism
- Clay on interactive elements creates a tactile UI language: glass surfaces, clay controls
- Per-page gradient palettes give each section personality while maintaining cohesion
- Bento grid replaces tabs as the primary navigation pattern for tool pages — big UX shift
- Comparison tray works like a shopping cart drawer — familiar pattern, glass-styled

</specifics>
