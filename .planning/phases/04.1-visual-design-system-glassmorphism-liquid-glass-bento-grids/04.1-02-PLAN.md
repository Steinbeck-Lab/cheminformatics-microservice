<task type="auto">
  <name>Task 1: Create BentoGrid and BentoCard reusable components</name>
  <files>frontend/src/components/common/BentoGrid.tsx</files>
  <action>
Create `BentoGrid.tsx` with two exported components:

**BentoGrid** (container):
Props: `{ pageKey: string; children: React.ReactNode; expandedId: string | null }`.
- Wraps children in a `LayoutGroup` with unique id based on pageKey (e.g., `id={`bento-${pageKey}`}`).
- Renders a CSS Grid container: `grid gap-4 grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4`.
- When `expandedId` is not null, switch to `grid-cols-1` to give expanded card full width.
- Apply glass-refraction-hover class to the grid container (desktop hover effect on child glass panels).

**BentoCard** (individual card):
Props:
```typescript
interface BentoCardProps {
  id: string;
  pageKey: string;
  name: string;
  description: string;
  icon: React.ComponentType<{ className?: string }>;
  size: BentoSize; // from bentoLayouts
  isExpanded: boolean;
  isOtherExpanded: boolean; // another card is currently expanded
  onToggle: () => void;
  children?: React.ReactNode; // the tool component rendered when expanded
}
```

Implementation:
- Use `motion.div` with `layoutId={`bento-${pageKey}-${id}`}` for FLIP animation.
- Spring transition: `{ type: "spring", stiffness: 300, damping: 25 }` (per user decision).
- When NOT expanded:
  - Apply size-based grid classes: hero = `col-span-2 row-span-2` (on lg+), medium = `col-span-2` (on sm+) or `col-span-1 row-span-2`, compact = `col-span-1`.
  - On mobile: all cards are `col-span-1` regardless of size.
  - Card shows: icon + name + description. Use `glass-bold rounded-2xl` styling.
  - Cursor pointer, onClick calls onToggle.
- When `isOtherExpanded && !isExpanded`: animate to `opacity: 0.3, scale: 0.95`, add `pointer-events-none`.
- When `isExpanded`:
  - `col-span-full row-span-full z-10` to fill grid.
  - Show card header (icon + name + description) at top.
  - Close button (X icon from lucide-react) in top-right corner using shadcn Button variant="ghost" size="icon".
  - Children (tool component) rendered inside `AnimatePresence` with `motion.div` that fades in: `initial={{ opacity: 0 }} animate={{ opacity: 1, transition: { delay: 0.2 } }} exit={{ opacity: 0 }}`.
  - After expand, call `scrollIntoView({ behavior: 'smooth', block: 'nearest' })` on the card element using a ref.

Use `useMediaQuery('(max-width: 639px)')` to force all cards to col-span-1 on mobile.

Do NOT nest glass layers (anti-pattern from research). The BentoCard is the glass surface; its children should use transparent backgrounds.

  <action>
Refactor ChemPage.tsx to replace the current sidebar-tab navigation with BentoGrid. This is a major refactor (~545 lines).

**Keep:**
- All 14 tool imports (StereoisomersView, DescriptorsView, etc.)
- All lucide icon imports
- The `tabs` array definition (id, name, description, icon, category, component)
- Page-level animation variants (pageVariants)

**Remove:**
- Sidebar rendering code (the left panel with category groups and tab buttons)
- LayoutGroup animated tab indicators (the sliding indicator bar)
- Category-based grouping in the sidebar
- The `categories` and `categoryOrder` constants (no longer needed -- bento grid shows all tools in a flat grid)
- sidebarItemVariant, sidebarStaggerContainer, contentVariants animation variants (replaced by BentoGrid animations)

**Add:**
- Import BentoGrid, BentoCard from `@/components/common/BentoGrid`
- Import getBentoSize from `@/config/bentoLayouts`
- Import GradientMesh from `@/components/common/GradientMesh`
- State: `const [expandedToolId, setExpandedToolId] = useState<string | null>(null)`
- URL param support: use `useParams` to read initial tool from URL (e.g., `/chem/descriptors`), set as initial expandedToolId. Use `useNavigate` to update URL on expand/collapse.

**Page structure:**
```tsx
<motion.div variants={pageVariants} initial="hidden" animate="visible" className="relative min-h-screen">
  <GradientMesh page="chem" />
  {/* Page header */}
  <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 py-8">
    <h1 className="text-3xl font-bold text-foreground">Chemical Analysis</h1>
    <p className="text-muted-foreground mt-2">Analyze molecules with descriptors, stereoisomers, similarity, and more.</p>
  </div>
  {/* Bento grid */}
  <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 pb-12">
    <BentoGrid pageKey="chem" expandedId={expandedToolId}>
      {tabs.map(tab => (
        <BentoCard
          key={tab.id}
          id={tab.id}
          pageKey="chem"
          name={tab.name}
          description={tab.description}
          icon={tab.icon}
          size={getBentoSize('chem', tab.id)}
          isExpanded={expandedToolId === tab.id}
          isOtherExpanded={expandedToolId !== null && expandedToolId !== tab.id}
          onToggle={() => {
            const newId = expandedToolId === tab.id ? null : tab.id;
            setExpandedToolId(newId);
            // Update URL
            navigate(newId ? `/chem/${newId}` : '/chem', { replace: true });
          }}
        >
          <tab.component />
        </BentoCard>
      ))}
    </BentoGrid>
  </div>
</motion.div>
```

Keep `useEffect` for reading `toolName` from URL params on mount and setting `expandedToolId`.
