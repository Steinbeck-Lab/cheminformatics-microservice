## Decisions Made
- glass-refraction-hover placed in @layer components (not @utility) because @utility cannot contain :hover pseudo-class or @media query wrappers
- Privacy and Terms pages use identical neutral gray gradient palettes (same values, separate config entries for extensibility)
- getBentoSize() returns 'compact' as default fallback for any tool ID not found in configuration
